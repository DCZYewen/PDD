# PDD
Primary Drive Display

这是车机的软件，含有下位机的协议和通信细节以及驱动，含有上位机的全套软件。

-----------

车机主要有三个功能，
- 车辆行驶状态监控，方便恶劣路况行驶和高原行驶。
- 车辆前后实时监控（可拓展为全方向），并录像。
- 车载娱乐，可以播放视频和音频。
